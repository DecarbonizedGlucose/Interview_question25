### 6.指针--好像看起来有点像，哪里错了呢？

请阅读以下代码，判断哪些操作不合法，并试着修改代码使程序可以编译通过吧

```c
#include <stdio.h>

int main() {
    int a = 100, b = 200, c = 300;
    
    const int *p1 = &a;          
    int * const p2 = &b;        
    const int * const p3 = &c;   

    // 操作列表
    *p1 = 110;  // ①
    p1 = &b;     // ②
    *p2 = 210;   // ③
    p2 = &c;    // ④
    *p3 = 310;  // ⑤
    p3 = &a;    // ⑥

    printf("a = %d, b = %d, c = %d\n", a, b, c);
    return 0;
}

```

```
A1：
数组指针是指向数组的指针，本质是指针
指针数组是里面元素都是指针的数组,本质是数组
函数指针是指向函数的指针,本质是指针

A2：
操作 1，4, 5，6 非法
1.p1 是一个指向常量的指针，不能通过 p1 来修改 a 的值
4.p2 是一个指针常量，它的指向不能改变。
5.p3 是一个指向常量的指针，不能通过 p3 来修改 c 的值。
6.p3 是一个指针常量，它的指向不能改变。


A3：
空指针是一个不指向任何有效内存地址的指针。它被用来表示一个指针没有被初始化，或者它所指向的内存已经被释放。
野指针是指向一块已释放或无效内存地址的指针。它是一个非常危险的概念，因为你无法通过简单的 if (p == NULL) 来判断它是否合法。
二级指针，也叫指向指针的指针，它存储的是另一个指针变量的地址。它允许你通过函数修改一个一级指针的指向。
```
